Come installare un pacchetto:
prima di installare un qualsiasi pacchetto è buona norma aggiornare il reposistory

sudo apt-get update

poi bisogna cercare il pacchetto

apt-cache searhc keyword

adesso si passa con l'installazione, infatti il comando precedente
darà il nome effettivo del pacchetto.

sudo apt-get install name

Tuttavia i pacchetti(scaricati nel sistema), possono essere aggiornati alla versione più recente:

sudo apt-get update aggiorna i reposistory
sudo apt-get upgrade aggoirna i pacchetti scaricati

Per eliminare i pacchetti vengono usati due comandi:

sudo apt-get remove name
sudo apt-get purge name

La differenza è quella che con remove elimina tutti i file tranne quelli di configurazione.
Purge elimina tutto.
